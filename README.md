# simple-rn-and

Just a simple React Native boilerplate, L2A.
Typescript, Android only, Hermes is on, apk split by abi.
Latest npm packages (nl-*.txt files).

Note for distros like MX Linux (Debian like distros, unchecked).
Insted of:
  yarn start
  yarn android
may run:
  ss/sss
